# astrbot-qq-say

可直接通过 AstrBot 插件仓库链接安装的独立插件仓库。

## 安装方式

### 方式一：通过 GitHub 链接安装

在 AstrBot 插件安装界面中填入本仓库链接即可。

### 方式二：手动安装

1. 克隆或下载本仓库。
2. 将仓库中的所有文件直接放入 AstrBot 的单个插件目录中。
3. 在 AstrBot 插件配置页填写所需配置。
4. 重启 AstrBot 或重载插件。

## 使用

- 主命令：`/qq说`
- 详细说明：见仓库内 `README.md` 下方内容

## 说明

- `metadata.yaml`、`main.py`、`_conf_schema.json` 已放在仓库根目录，兼容 AstrBot 链接安装。
- 已将本地敏感 API 地址和 Key 替换为占位内容（如适用）。
- 不包含运行环境中的本地配置文件。

# QQ说 插件

生成一条或多条仿 QQ 合并转发聊天记录。

## 安装

1. 将仓库中的所有文件直接放入 AstrBot 的单个插件目录中。
2. 在 AstrBot 插件配置页按需修改分隔符、白名单和保护账号配置。
3. 重启 AstrBot 或重载插件后即可使用。

## 命令

- `/QQ说 @某人 文本`
- `/qq说 @某人 文本`
- `/QQ说 @某人1 文本1 * 文本2\n第二行继续同一条消息\n---\n@某人2 文本3 * 文本4`

## 规则

- `message_separator`：同一发言人的多条消息分隔符，默认 `*`
- 普通换行：继续沿用上一位发言人，并作为同一段内容续写
- `speaker_separator`：表示下一段开始切换发言人，默认 `---`；既可以单独占一行，也可以直接写成 `### @某人 文本`
- 只有第一段或切换发言人的那一行需要以 `@某人` 开头

## 群聊白名单

- `group_whitelist` 留空：所有群都可以使用
- `group_whitelist` 填入群号：只有白名单中的群可以使用
- 支持用英文逗号 `,`、中文逗号 `，`、空格或换行分隔多个群号
- AstrBot 管理员不受群聊白名单限制

## 私聊开关

- `allow_private_chat` 默认关闭
- 开启后才允许在私聊中使用 QQ说
- AstrBot 管理员不受私聊开关限制

## 保护账号

- `protected_users` 默认保护 `3983931436`、`1332053049`
- `protected_users` 留空：不限制被伪造目标
- `protected_users` 填入 QQ 号：普通用户无法伪造这些账号的消息
- 支持用英文逗号 `,`、中文逗号 `，`、空格或换行分隔多个 QQ 号
- AstrBot 管理员不受保护账号限制

## 行为

- 自动读取被 @ 群友的 QQ 号和名字
- 支持直接输入 `@QQ号`，插件会尽量查询并补全对应昵称
- 输出为 QQ 合并转发消息节点，不再生成截图图片
- 分隔符可在插件配置中修改
- 主要用于娱乐用途
